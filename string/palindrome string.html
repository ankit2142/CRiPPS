Palindrome String 

       
        
        Given a string S, check if it is palindrome or not. 

        Example 1:

        Input: S = "abba"
        Output: 1
        Explanation: S is a palindrome.

        Example 2:

        Input: S = "abc" 
        Output: 0
        Explanation: S is not a palindrome
        
        

        Code
               
        int isPlaindrome(string s)
	    {
	     
	    int i=0,j=s.size()-1;
	    while(i<j)
	    {
	        if(s[i]!=s[j])
	         {
	             return 0;
	         }
	         i++;
	         j--;
	    
	     }
	     return 1;
	    }
